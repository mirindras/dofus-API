# dofus-API
 La-chasse-aux-monstres

## Install

extraire dans votre PC le fichier " dofusAPI.zip " et puis entrer le commande ci desous dans votre terminale
```
bundle install
```

## Listes des gems


>watir

>selenium-webdriver

>json

>httparty

## Utilisation

entrer dans votre terminal 
```
application.rb
```
pour lancer le programme
